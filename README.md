Per compilare usare
```
clang++ -std=c++11 -stdlib=libc++ -Weverything main.cpp -o main
```
per fare build e creare main.exe, lancia **f6** da *VsCode*

per lanciare il programma, devi aprire il terminale ed eseguire
```
cd src
./main.exe instance01
```
